# ブラウザゲーム制作ページ コンテンツ骨格

**対象ファイル:** `pages/business-game-dev.php`  
**目的:** ブラウザゲームを作成してほしいという顧客を集められるページにする。

---

## 目的・ターゲット（ページ設計の前提）

### 目的
- ブラウザゲームの受託制作を依頼したい顧客の獲得

### ターゲット
- **メイン:** Steam などのハイスペックゲームではなく、**ブラウザから気軽にプレイできるゲーム**を作りたい企業
- **利用シーン:**
  - 企業の新規顧客キャンペーン
  - 既存顧客のリテンション・解約防止・キャンペーン（プレゼント配布など）
  - 公共団体の観光地紹介・市民への情報通知 など
- **外国人向け:**
  - 事業の海外・外国人向け展開をしたい企業・団体
  - 飲食店・サービス店舗で外国人顧客へのアピールにゲームを活用したい
  - 観光地の情報を外国人に発信したい（多言語・英語での案内）
  - 外国人向けサービス展開でゲームプロモーションをしたい

## 1. 作成できるゲーム概要

### 1-1. おすすめのゲームタイプ

**a. 動作環境**
- ブラウザで動作（PC・スマートフォン対応）
- 利用はスマートフォンが中心のため、**スマホ縦型**で快適に動くゲームを中心に制作

**b. プレイ設計**
- 複雑な操作やパラメーターは設けず、**基本的にタップのみ**で操作
- 平均プレイ時間は**約10分**を想定した設計

**c. 外国人・英語向け対応（オプション）**
- **全コンテンツを英語で作成**（UI・テキスト・説明文など）
- **音声での英語読み上げ**対応
- **チュートリアルも英語**で制作（操作説明・ルール説明など）
- **CrazyGames 等の世界的なゲームプラットフォームへの掲載サポート**（実績あり）
  - 海外ユーザーへのリーチ拡大、ブランド認知に活用可能

### 1-2. ゲームジャンル（対応例）

| ジャンル | 説明・例 |
|----------|----------|
| パズル系 | マッチング、並べ替え、消す系など |
| 簡易アクション系 | 避ける、集める、タップで反応するなど |
| 音ゲー | リズムに合わせたタップ・簡易演奏 |
| 占い系 | 診断・結果表示・シェア誘導 |
| ノベルゲーム系 | 選択肢分岐・ストーリー体験・情報案内 |

※上記を組み合わせた複合タイプも可能（例：ノベル＋ミニゲーム）

## 3. ゲーム作成のこだわり

- **ビジネスやゲームに合った魅力的なキャラクター**の作成
- **楽しくプレイできるUI・音楽・SE**の設計
- **ビジネス目的を達成するためのCVへの自然な誘導**（無理のない導線）
- **オリジナル仕様による「フック」**（他では味わえない違和感・特別感）
- **小規模でも楽しめるゲーム設計**（短時間で達成感を感じられる）
- **魅力的に見える世界観の構築**（ストーリー・ビジュアル・トーン）
- **外国人向け対応** … 全編英語・英語音声読み上げ・英語チュートリアルで、海外・インバウンド向けにそのまま利用可能

### 3-2. AI利用についての考え方

- AIで素材を作成する場合、**トラブル防止のため**納品時に以下を説明した**資料を添付**する  
  … 利用したAIツール、公開の権利、所有権など
- AI素材を作成する場合は、**参照元となる素材は版権があるもののみ**を使用する
- **偶然の重複・類似を防ぐ**ため、納品時に**発注者・製作者の両者で素材のチェック**を行う（AI生成と既存著作物の重複・類似がないか確認）

## 4. 料金

| プラン | 価格 | 想定プレイ時間 | 利用イメージ | ゲームルール | 外部連携 |
|--------|------|----------------|--------------|--------------|----------|
| **mini** | 30万円 | 約10分 | 1〜2回プレイ、単純なキャンペーン向け | 1ルール（合わせて消す／タップで保管／避ける等） | なし |
| **standard** | 100万円 | 20分〜 | 繰り返し遊べる、情報を知ってもらう・紹介する | 2〜3ルール（トーク＋プレイ／3タイプのゲーム／スコアで何か起こる等） | あり（行動を別サーバーへ CSV/JSON 等） |
| **full** | 200万円〜 | 30分〜 | 1週間〜1ヶ月計画で遊ぶ、単体で魅力のあるゲーム | 複数ルール、没入型（OP・チュートリアル・プレイ・ED 等） | あり（行動連携＋通知・情報配信）、App Store 等・**CrazyGames等の世界ゲームプラットフォーム掲載サポート**も |

### 4-2. 予算の考え方

- 予算は**ゲームの規模**でおおむね決まりますが、**ブラッシュアップをどこまで行うか**によっても作業時間が変わり、**予算が変動**します。
- そのため、**一定の完成状態まで制作した時点**で、**予算を追加してブラッシュアップするかどうか**をご確認します。
- ゲームはシステムやWebサイトと異なり**「完成」の定義がしづらい**ため、**フェーズ分け・予算・完成タイミング**を確認しながら進めます。

### 4-3. ゲーム作成のステップ

1. **ヒアリング・方向性の確認・予算提示**  
   → この時点で**紙芝居的な画面イメージ**を作成する
2. **ゲームの初期バージョンを作成**  
   → 画像・UI・音などは**仮のもの**で制作
3. **ブラッシュアップ範囲の説明・要望のご確認**  
   → どの部分を詰めるかを共有し、ご要望を確認
4. **ブラッシュアップ・確認**  
   → 1〜2回程度のやりとりで仕上げ
5. **テスト**
6. **リリース**  
   → リリース場所の確認・配置
7. **リリース後のバージョンアップ・更新**  
   → 別途対応。**JSON等の更新でゲーム内容が変わる仕組み**にしておき、更新コストを下げる方法もあります。

## 5. プラン・問い合わせ（3パターン）

ターゲットの**発注意思・状態**で3つに分け、それぞれに合った入口を用意する。

### 5-A. 企画から相談したい（何から始めればいいかわからない）

- **対象:** ゲームには興味があるが、何から始めていいかわからない。使い方や企画から相談したい。
- **フォーム・導線:**
  - フリーテキストで相談内容を入力
  - （任意）背後でAIを使った対話型で状態をヒアリングし、送信しやすい形にする
  - **予算が数万円程度でないこと**を確認し、足切り
- **受付後:** 内容を社内で精査し、MTGスケジュールを確定

### 5-B. 企画・イメージがある（プロモのイメージや企画書がある）

- **対象:** ゲームを使ったプロモのイメージや企画書がある。どんなゲームにして、どう誘導・情報収集・CVさせるか相談したい。
- **フォーム・導線:**
  - 企画書・考えていることを**投げ込めるUI**（画像ファイル等も可）
  - AI対話で具体化し、問い合わせしやすい形にして送信
- **受付後:** 内容を社内で精査し、MTGスケジュールを確定

### 5-C. 作りたいゲームが決まっている（キャラ・絵があり構築したい）

- **対象:** 作りたいゲームが決まっている。キャラクターや絵があり、ゲームとして構築したい。
- **フォーム・導線:**
  - Bと同様に、**より具体的なゲームの詳細**を入力（動作環境・求める品質・仕様など）
  - 作れないレベルのものではないか判断するための情報を収集
- **受付後:** 内容を社内で精査し、MTGスケジュールを確定

## 7. Whatever Partners（WP）のゲーム制作の立ち位置

- **伝えたいこと:** ゲーム業界のレイヤーを理解してもらい、**得意分野と価格の安さ**で納得して問い合わせしてもらう。

### ゲーム業界のレイヤー（上から下のイメージ）

1. **コンシューマー系の大企業ゲーム制作会社** … 据置・大作タイトル
2. **Steam等・3D描画エンジンのハイスペックゲーム** … PC/Steam向け
3. **iPhoneアプリ等・単品購入・ゲーム内課金** … ゲーム好きが納得する品質
4. **★WPの位置づけ** … **JSゲームエンジンでブラウザ上の軽量ゲーム**。多彩な表現・キャラクター・音楽・SEで、Webサイト訪問者には魅力的なコンテンツ。ただし「ゲーム単体で長時間遊びたい人」にはやや物足りない層。
5. **同人ゲーム制作チーム** … ビジネス目的より「自分たちが面白いものを作る」
6. **ブラウザ上の簡易JSゲーム** … 表現力・世界観は控えめなケースが多い

### WPの強み（ページで訴求する文言のたたき）

- ブラウザで動く**軽量**でありながら、**Webサイトより表現力が高く**、訪問者にとって**魅力的なコンテンツ**にできる
- **ビジネス目的**（集客・リテンション・観光・教育・キャンペーン）に特化した設計が得意
- **価格を抑えた**プラン（mini 30万円〜）で、企画から制作まで一貫して対応
- **外国人向け** … 英語コンテンツ・音声読み上げ・英語チュートリアル・CrazyGames等の世界プラットフォーム掲載サポート（実績あり）で、インバウンド・海外展開を支援
